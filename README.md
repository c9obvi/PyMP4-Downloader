# MP4-DL
![](https://maeebnvejslkonktbeha.supabase.co/storage/v1/object/public/cdn/PyVideo.webp)
## description:
this is a quickstart script using existing libraries
to download "most" video URL's. This can be baked into other programs.

## Requirements:
```bash
pip install yt-dlp
```
